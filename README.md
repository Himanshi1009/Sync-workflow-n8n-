# n8n Sync Workflow

Automatically syncs all changes from the **Master Workflow** to all the **Target Workflows**.
> Except the credentials, everything in the workflow is synced - the nodes, connections, parameters and settings.

# Usage
1. Make changes in Master Workflow
2. Run the Sync Workflow
3. All targets updated 

# Configuration
Add/remove target node ids in the Config node.
Add the master node id in the Config node.
